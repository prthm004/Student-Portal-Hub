CS301: Algorithms & Data Structures

Course Overview

This course covers the design and analysis of data structures and algorithms. Topics include: abstract data types, stacks, queues, linked lists, trees, graphs, hash tables, sorting and searching algorithms, and time complexity analysis.

Learning Outcomes
  • Implement and apply various data structures
  • Analyze algorithm performance and complexity
  • Design efficient algorithms for problem-solving
  • Compare algorithmic approaches for optimization
